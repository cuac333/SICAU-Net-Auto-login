# 校园网自动登录脚本

四川农业大学校园网自动登录 + 热点保活工具。

## 功能

- 自动检测校园网认证状态，断线自动重连
- 热点保活：网络正常时自动保持热点开启
- 运行时可输入 `y` 开启热点保活，输入 `n` 关闭
- 账号密码保存到本地配置文件，首次运行后无需重复输入

## 使用方法

### 1. 安装依赖

```bash
pip install requests
```

### 2. 启动程序（二选一）

- 双击 `启动校园网.bat`
- 命令行运行 `python campus_login.py`

### 3. 首次运行

按提示输入校园网账号、密码，并选择是否开启热点保活（Y/N）。

## 注意事项

- `campus_net.conf` 包含个人账号密码，**严禁上传到公共仓库**。`.gitignore` 已默认排除此文件。
- 仅限校内使用，遵守学校网络管理规定。
- 需要已连接校园网 WiFi 环境。

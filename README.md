# 四川农业大学校园网自动登录 + 热点保活工具

## 功能

- 自动检测校园网认证状态，断线自动重连
- 热点保活：网络正常时自动保持热点开启
- 运行时可输入 `y` 开启热点保活，输入 `n` 关闭
- 账号密码保存到本地配置文件，首次运行后无需重复输入
- **原始错误提示**：登录失败时直接显示服务器返回的原始错误信息，不做映射转换，信息更准确
- **文件日志系统**：详细日志写入 `campus_login.log`，支持日志轮转（单文件 5MB，自动保留 3 个备份），便于排查问题

## 环境要求

- **Python 3.x**（推荐 3.7 及以上版本）

## 使用方法

### 1. 安装依赖

```bash
pip install requests
```

### 2. 启动程序（二选一）

- 双击 `启动校园网.bat`
- 命令行运行 `python campus_login.py`

### 3. 首次运行

按提示输入校园网账号、密码，并选择是否开启热点保活（Y/N）。

### 4. 创建桌面快捷方式

建议将 `启动校园网.bat` 的**快捷方式**放到桌面，方便日常使用：

1. 右键点击 `启动校园网.bat` → 发送到 → 桌面快捷方式
2. 之后双击桌面上的快捷方式即可启动

> **注意：是创建快捷方式，不要直接移动或复制 bat 文件本身。** 因为 `启动校园网.bat` 使用相对路径定位 `campus_login.py` ，如果移动了 bat 文件的位置，程序将无法找到这些文件而报错。快捷方式不会改变原文件位置，所以不受影响。

### 5. 设置开机自启

双击运行 `设置开机自启.bat`，会自动在 Windows 启动文件夹中创建快捷方式，开机后自动运行。

也可以手动设置：按 `Win+R`，输入 `shell:startup` 回车，将 `启动校园网.bat` 的快捷方式复制到打开的文件夹中即可。

## 文件说明

| 文件 | 说明 |
|------|------|
| `campus_login.py` | 主程序 |
| `启动校园网.bat` | 一键启动批处理文件 |
| `设置开机自启.bat` | 设置开机自启动 |
| `open_hotspot.ps1` | 打开热点脚本 |
| `close_hotspot.ps1` | 关闭热点脚本 |
| `campus_net.conf` | 配置文件（首次运行自动生成，含账号密码） |
| `campus_login.log` | 运行日志（自动生成，5MB 轮转，已加入 `.gitignore`） |

## 注意事项

- `campus_net.conf` 包含个人账号密码，**严禁上传到公共仓库**。`.gitignore` 已默认排除此文件。
- **配置文件为只读**：`campus_net.conf` 生成后会被设为只读属性，防止误修改。如果首次运行时输入的账号密码有误，或者需要修改密码，**请删除 `campus_net.conf` 文件后重新运行程序**，会提示重新输入账号密码并生成新的配置文件。
- `campus_login.log` 为运行日志，包含详细调试信息，已加入 `.gitignore` 自动忽略。日志单文件最大 5MB，超出后自动轮转，最多保留 3 个历史备份。
- 仅限校内使用，遵守学校网络管理规定。
- 需要已连接校园网 WiFi 环境。
- **不要移动 `启动校园网.bat` 的位置**，它通过相对路径找到同目录下的 `campus_login.py` 和 PowerShell 脚本，移动后程序将无法正常运行。如需在桌面使用，请创建快捷方式。

## 日志说明

程序采用双通道日志输出：

| 输出目标 | 级别 | 内容 |
|----------|------|------|
| 控制台（命令行） | INFO | 精简信息：登录成功/失败、关键错误提示 |
| 文件（`campus_login.log`） | DEBUG | 完整细节：IP 地址、请求参数、响应状态码、原始 HTML 等 |

登录失败时，控制台会显示类似 `❌ 登录失败：由于你登录失败次数较多,被禁用1分钟` 的原始服务器错误信息；如需排查具体原因，可打开 `campus_login.log` 查看完整服务器响应内容。
